# ☕ Campus Cafe Kiosk – C# Windows Forms Application

## 📌 Overview
The **Campus Cafe Kiosk** is a simple C# Windows Forms application that simulates a coffee ordering system.  
It allows users to enter their name, choose a syrup flavor, and optionally upgrade to a premium drink.  

The system demonstrates key object-oriented programming concepts such as **inheritance**, **method overriding**, and **basic GUI interaction**.

---

## 🎯 Features
- Capture customer name and syrup flavor
- Generate formatted order ticket
- Calculate drink price based on selection
- Upgrade option for premium drinks
- GUI-based interaction using Windows Forms
- Displays results and confirmation message

---

## 🧠 Concepts Used
- Inheritance (`DrinkOrder` → `PremiumDrink`)
- Method Overriding (`CalculateTotal()`)
- String Manipulation (`Trim()`, `Replace()`, `ToUpper()`)
- Conditional Logic (CheckBox)
- Windows Forms UI (TextBox, Label, Button, CheckBox)
- MessageBox for user feedback
